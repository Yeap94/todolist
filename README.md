# Todolist application
This application powered by angularJS 1.7.9 with typescript and webpack

# Installation

## NodeJS

[source](https://nodejs.org/en/)

```
node -v
v12.16.3

npm -v
6.14.4
```

# Install application dependencies

```
npm i
```

# Building

Run `npm run build` to build the project.


# Running

```
npm start
```
